# Weather Web App

## Setup and Running
1. Clone the repository:
   ```bash
   git clone https://github.com/USERNAME/REPOSITORY_NAME.git
   cd REPOSITORY_NAME
   
2. Add your API key:
Insert your own OpenWeather API key in the script.js file at the line const key = 'YOUR_API_KEY';.

3. Run the application:
Open the index.html file in your browser.
